About

The table and network above depict the 260 phages described in Moriniere et al. 2026. The network is a force directed graph based on vcontact connections between phage, and the table shows basic metadata and experimental results for each phage.

  • Click on any phage in the network or table to explore detailed information about its genome structure, receptors, and characteristics
  • Use the brush tool in the network graph to select and filter phages in the table, or click the magnifying glass to search or filter in the table
  • Hover over table rows to highlight corresponding nodes in the network
  • Follow the link below the network or in the phage popups to explore the RB-TnSeq data in a genome browsers

Links directly to genome browsers:
